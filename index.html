Gestión de Patentes Municipales

2023

Desarrollé un programa que permitía gestionar las patentes municipales de la municipalidad de El Colmo. Este proyecto incluyó un menú interactivo, validación de datos de patentes, cálculo de estadísticas descriptivas y la generación de archivos .txt. Se utilizaron clases como Empresa, Patente, Fecha y Rut, además de aplicarse estructuras de datos como colas para gestionar los datos de manera eficiente.

  • Generación de archivos y reportes
  • Validación y calculo automático
  • Gestión de datos eficiente
